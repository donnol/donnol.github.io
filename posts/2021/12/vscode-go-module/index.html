    vscode-go在go.mod在非根目录情况下失效的问题

    December 20, 2021
    vscode-go
    gomod

    问题如图：

    解决：

    添加配置：

    {
    // ...
   "gopls": {
        "experimentalWorkspaceModule": true
    },
    // ...
}

    等go 1.18的workspace模式推出之后，应该就不需要配置这个了。
